# TaskManager

TaskManagerは、シンプルかつモダンなUIでタスク管理ができるWebアプリです。

## 特徴
- タスクの追加・完了・削除が可能
- ローカルストレージに自動保存
- スタイリッシュなデザイン
- サンプルタスク10件を初期表示

## 使い方
1. `index.html` をブラウザで開く
2. 新しいタスクをフォームから追加
3. タスクの完了/未完了切替や削除もワンクリック

## ファイル構成
- `index.html` : アプリ本体のHTML
- `style.css` : デザイン用CSS
- `app.js` : 機能実装用JavaScript
